Model blokowy CSS

Wszystkie elementy projektowane w HTML można traktować jak odpowiednio sformatowane i rozmieszczone na stronie prostokątne bloki.

Każdy z takich bloków posiada:

  • margin - margines zewnętrzny, pusty obszar wokół ramki, nie ma koloru tła i jest przeźroczysty
  • border - obramowanie wokół zawartości elementu, ma styl i kolor
  • padding - odstęp pomiędzy obramowaniem a zawartością elementu
  • content - zawartość elementu
Obramowanie

Obramowanie

Szerokość obramowania border-width

border-width: wartość;

Ustawienie atrybutu:

  • 1 wartość - jednakowa szerokość dla wszystkich krawędzi
  • 2 wartości - ustawienie szerokości dla kolejno obu poziomych i obu pionowych krawędzi
  • 3 wartości - pierwsza wartość określa szerokość krawędzi górnej, druga krawędzie pionowe, a trzecia krawędzi dolnej
  • 4 wartości - góra, prawo, dół, lewo

Szerokość krawędzi może być definiowana w dowolnych jednostkach lub za pomocą trzech wartości:

  • thin
  • medium
  • thick

Styl obramowania

Podobnie jak w przypadku definiowania szerokości możemy definiować jedną, dwie, trzy lub cztery wartości oddzielnie.

Wartości atrybutu border-style:

  • none - brak obramowania
  • hidden - obramowanie ukryte
  • dotted - obramowanie kropkowane
  • dashed - obramowanie kreskowane
  • double - podwójny
  • groove - 3D rowki
  • ridge - 3D grzbiety
  • inset - ramka
  • outset - odwrotna ramka

Aby ramka była widoczna podajemy styl i szerokość.


Kolor obramowania

Podobnie jak w przypadku innych atrybutów możliwe jest podanie jednej, dwóch, trzech lub czterech wartości, lub zdefiniowanie kolorów dla poszczególnych krawędzi.

Jako wartości koloru można używać:

  • Kodu heksadecymalnego
  • Kodu RGB
  • Predefiniowanych nazw kolorów

Przykład:

Lorem ipsum dolor sit amet.

Lorem ipsum dolor sit amet.

Lorem ipsum dolor sit amet consectetur.

Lorem ipsum dolor sit amet consectetur.

Lorem ipsum dolor sit amet.

Officia consequuntur pariatur consectetur saepe.

Porro laborum reiciendis ea dolor!

Marginesy

Marginesy

Margines zewnętrzny (margin)

Margines zewnętrzny określa przestrzeń wokół definiowanego elementu. Przestrzeń ta oddziela dany element od innych elementów strony.

selektor {
            margin-top / bottom / left / right: wartość;
         }

Szerokości marginesów zewnętrznych możemy definiować pojedyńczo dla każdej strony, całościowo dla wszystkich lub też dla wszystkich, podając jedną, dwie, trzy lub cztery wartości.

Rozmiar marginesu podawany jest w jednostkach długości (px, cm, in) lub jako rozmiar auto

Margines wewnętrzny (padding)

Margines wewnętrzny to odległość pomiędzy zawartością elementu a jego obramowaniem.

selektor {
            padding-top / bottom / left / right): wartość;
         }

Podobnie jak dla marginesów zewnętrznych można definiować rozmiary wszystkich stron marginesu lub każdą oddzielnie.

Przykład:

Lorem, ipsum.

Lorem, ipsum dolor.

Ex, temporibus molestiae.

Vel, praesentium modi!

Aliquid, itaque maiores.

Obrys

Obrys

Obramowanie wokół elementu może być tworzone poprzez zdefiniowanie obrysu.

Obrys różni się od obramowania tym że:

  • nie zajmuje miejsca, przez co nie wpływa na rozmiar i położenie elementu
  • jest stworzony na wierzchu elementu
  • jest definiowany dla wszystkich krawędzi jednocześnie
selektor { 
            outline: wartości;
         }

Możemy w tym atrybucie określić kolor, styl i szerokość obrysu.

Żeby obrys wyświetlał się, obowiązkowo musimy podać styl.

Przykład

Outline:

Lorem ipsum dolor sit amet.



Border:

Lorem ipsum dolor sit amet.

Rozmiary elementów

Rozmiary elementów

W modelu blokowym można ustalić dokładne rozmiary różnych elementów, na przykład akapitu, diva, itp.

Atrybuty rozmiaru:

  • width - szerokość
  • min-width - szerokość minimalna
  • max-width - szerokość maksymalna
  • height - wysokość
  • min-height - wysokość minimalna
  • max-height - wysokość maksymalna

Przykład:

height: 200px

height: auto

Przepełnienie

Przepełnienie

Jeżeli zawartość elementu nie mieści się w podanych rozmiarach możliwe jest ukrycie niemieszczącej się zawartości, powiększenie rozmiaru elementu lub wyświetlenie suwaków, które pozwolą na obejrzenie całej zawartości elementu.

selektor {
            overflow: visible / hidden / scroll
         }

Możliwe wartości atrybutu overflow:

  • visible - niezależnie od rozmiaru rodzica, element jest widoczny
  • hidden - treść niemieszcząca się w zdefiniowanym rozmiarze zostaje ukryta
  • scroll - powoduje przepełnienie i dodanie paska przewijania, który pozwoli na obejrzenie zawartości wewnątrz bloku, pasek zostaje dodany w pionie i poziomie
  • auto - automatycznie, jeżeli jest to konieczne, suwaki zostają wyświetlone

Przykład:

overflow: visible

overflow: scroll

overflow: hidden

overflow: auto
