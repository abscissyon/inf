Styl listy

Typ punktora

Składnia:

 selektor {
            list-style-type: typ;
          }

Typy punktorów:

  • disc - koło
  • circle - okrąg
  • square - kwadrat
  • decimal - liczby arabskie
  • lower-roman - liczby małe rzymskie
  • upper-roman - liczbe duże rzymskie
  • lower-alpha - małe litery
  • upper-alpha - duże litery
  • none - brak punktora

Selektorami mogą być znaczniki definiujące listy (ul, li)

Parametr type określa wygląd punktora

Pozycja punktora

Oprócz typu punktora można również ustawić jego położenie względem tekstu.

Składnia:

selektor {
            list-style-position: inside/outside;
         }

Wartości atrybutu:

  • inside - punktor pojawia się wewnątrz zawartości
  • outside - punktor pojawia się na zewnątrz zawartości/li>

Podpis

W celu określenia położenia podpisu dodanego do tabeli używamy atrybutu caption-side.

Składnia:

selektor {
            caption-side: wartości;
         }

Możliwe wartości:

  • top
  • bottom
  • left
  • right

Można określić również sposób obramowania dla pustych komórek.

selektor {
            empty-cells: show/hide;
         }

Table layout

Do rozplanowania tabeli na stronie można użyć atrybutu table-layout

Atrybut ten przyjmuje następujące wartości

  • auto - szerokość tabeli jest ustawiana automatycznie i zależy od zawartości
  • fixed - szerokość tabeli jest stała i wynika z zadeklarowanej szerkości poszczególnych komórek oraz szerokości całej tabeli

Odstępy między komórkami

Odstęp podajemy w jednostkach długości. Można podać jedną lub dwie wartości.

Składnia:

selektor {
            border-spacing: odstęp;
         }

Szerokość i wysokość tabeli

Składnia

selektor {
            height: wartość;
            width: wartość;
         }

Zadanie

Zaprojektuj tabelę w której szerokość całkowita tabeli będzie wynosiła 100%.

Formatowanie znaczników table, td i tr: obramowanie 2 piksele, linia ciągła, kolor niebieskim, obramowanie pojedyńcze i rozmieszczenie z wartością fixed.

Formatowanie dla znacznika th: obramowanie 2 piksele, linia ciągła, kolor czerwony, obramowanie pojedyńcze, wysokość 50 pikseli.
